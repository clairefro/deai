## Using Ollama

1. download Ollama

(mac)
`brew install ollama`

2. download a model (for example, gemma3 ~3.3G)

`ollama pull gemma3`

3. run ollama server locally

either in background :
`ollama serve > /dev/null 2>&1 &`

or in open terminal tab:
`ollama serve`
